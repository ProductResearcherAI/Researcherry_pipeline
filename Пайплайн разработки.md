# Роадмэп разработки ResearcherryAI

Утверждено СЕО проекта, Cyrill Kotilevskii.

## Базовая платформа: Continue.dev

**Техническое решение**: Используем [Continue.dev](https://docs.continue.dev/) как базовую open-source платформу для AI-агентов вместо разработки с нуля.

**Преимущества Continue.dev**:
- Open source проект с активным сообществом
- Встроенная поддержка множества AI провайдеров (Anthropic, OpenAI, Google, Open Router, Windsurf)
- Система MCP серверов для интеграций
- Каталог промптов и возможность создания своих правил (.uml файлы)
- Готовая архитектура для форка и кастомизации
- Возможность интеграции существующего ProductResearcherAI агента как дефолтного

## Ценовая модель

**Freemium подход**:
- **Бесплатные пользователи**: Google Flash (бесплатный API) с ограничениями на количество запросов
- **Индивидуальные платные**: GPT-4 и другие модели средней стоимости
- **Корпоративные клиенты**: Claude Sonnet (Anthropic) - премиум модель для корпораций
- **Enterprise**: Выделенные ресурсы и кастомные интеграции

## Техническая архитектура

**Базовый стек**:
